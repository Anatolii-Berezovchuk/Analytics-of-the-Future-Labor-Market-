Это означает, что кластеризация действительно выделила **реальные группы профессий**, отличающиеся по влиянию ИИ.

## **4.4. Корреляционный анализ (Spearman)**

| Пара переменных | rho | p-value | Значимость | Интерпретация |
|-----------------|------|---------|------------|----------------|
| automation_risk_percent ↔ ai_replacement_score | **0.969** | 0.0000 | Значимо | Очень сильная положительная связь |
| automation_risk_percent ↔ salary_change_percent | -0.002 | 0.774 |  Нет | Связи нет |
| ai_replacement_score ↔ salary_change_percent | -0.006 | 0.467 |  Нет | Связи нет |
| automation_risk_percent ↔ skill_gap_index | 0.010 | 0.239 |  Нет | Связи нет |

**Главный вывод по корреляциям:**

- Между **automation_risk_percent** и **ai_replacement_score** связь **очень сильная** (rho ≈ 0.97).  
  Это означает, что оба показателя фактически измеряют *один и тот же эффект* — риск замещения ИИ.

- Ни один показатель риска ИИ **не связан** с изменением зарплат.  
  То есть высокий риск автоматизации **не означает** падение зарплаты.

- Skill gap также **не связан** с риском автоматизации.
